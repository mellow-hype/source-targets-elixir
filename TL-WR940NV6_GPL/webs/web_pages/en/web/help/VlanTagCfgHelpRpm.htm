Aide IPTV

IPTV (Internet Protocol Télévision) est un système par lequel les services de télévision sont acheminés par une suite de protocoles internet via un réseau à commutation de paquets tels qu'un réseau local ou internet.

Proxy IGMP - Sélectionner pour activer la fonction Proxy IGMP si le service IPTV du FAI est basé sur IGMP.

IPTV - Sélectionner pour activer la fonction IPTV si le service IPTV du FAI n'est pas basé sur IGMP.

Profile ISP - Sélectionner le profil FAI (Fournisseur d'Accès à Internet) approprié pour réaliser la configuration. Il existe trois types de profils :

1. Country-ISP Name - Les profils FAI pré paramétrés sont au format "Pays-FAI". Trouvez et sélectionnez votre FAI depuis la liste déroulante.

2. Pont - Si votre FAI n'est pas listé et qu'aucun autre paramètre n'est requis pour paramétrer l'IPTV, vous pouvez sélectionner ce profil et définir le port dédié à l'accès IPTV.

  • LAN 1/2/3/4 - SpecifySpécifier quel type de données le port LAN transmettra. Par exemple, si IPTV est utilisé pour le port LAN1, ce dernier ne transmettra que les données IPTC et vous devrez brancher le décodeur au port LAN1.

3. Autre - Si votre FAI n'est pas listé mais qu'il vous à communiqué les paramètres requis pour Internet, la VoIP l'IPTV (N° de VLAN et priorités) , sélectionnez ce profil.

  • Priorité Internet/VoIP/N° de VLAN IPTV - Configurer les ID et priorités de VLAN comme indiqué par le FAI.
  • Tag 802.11Q - Choisir si vous taguez les paquets internet selon 802.11Q.
  • N°de VLAN du Multicast IPTV/Priorité - Vous pouvez activer la fonction Multicast IPTV selon votre convenance, et configurer les N°de VLAN et priorités selon les indications du FAI.
  • LAN 1/2/3/4 - Spécifier quel type de données le port LAN transmettra. Par exemple, si IPTV est utilisé pour le port LAN1, ce dernier ne transmettra que les données IPTC et vous devrez brancher le décodeur au port LAN1.

Cliquer sur Sauvegarder pour enregistrer tous vos paramètres.
