Aide Statistiques sans fil

Cette page affiche, pour chaque station sans fil connectée, les paramètres suivants : Adresse MAC, État actuel, Paquet(s) reçu(s) et Paquet(s) envoyé(s).

  • Adresse MAC - Adresse MAC de la station sans fil connectée.
  • État actuel - État actuel de la station sans fil connectée : STA-AUTH / STA-ASSOC / STA-JOINED / WPA / WPA-PSK / WPA2 / WPA2-PSK / AP-ACTIVÉ / AP-DÉSACTIVÉ / Déconnecté.
  • Paquet(s) reçu(s) - Paquets reçus par la station.
  • Paquet(s) envoyé(s) -Paquets envoyés par la station.
  • Configurer - Ce bouton permet de charger un élément dans la liste Filtrage MAC sans fil.
    • Refuser - Refuse l'accès de la station si la fonction Filtrage MAC sans fil est activée.
    • Autoriser - Autorise l'accès de la station si la fonction Filtrage MAC sans fil est activée.

Pour rafraîchir cette page et afficher les stations sans fils connectées, cliquez sur le bouton Actualiser.

Si la liste occupe plus d'une page, utilisez le bouton Suivant ou Précédent pour afficher la page souhaitée.

Remarque : Cette page est actualisée automatiquement toutes les 5 secondes.
