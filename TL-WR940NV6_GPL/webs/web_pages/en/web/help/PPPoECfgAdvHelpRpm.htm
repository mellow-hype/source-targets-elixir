Aides Paramètres PPPoE avancés

Taille MTU - La taille MTU (Maximum Transmission Unit) par défaut est de 1480 octets, ce qui est en général suffisant. Cependant, en fonction de votre FAI, il se peut que vous deviez la changer. Ne modifiez cette valeur que si nécessaire.

Nom de service/nom CA - Ne créez les noms de service et de CA que si cela est requis par votre FAI.

Adresse IP spécifiée - Si votre FAI ne transmet pas automatiquement l'adresse IP au routeur lors de la connexion, cliquez sur « Utiliser l'adresse IP spécifiée par le FAI » et saisissez l'adresse IP en notation décimale pointée fournie par votre FAI.

Intervalle de détection en ligne - La valeur par défaut est 0. Vous pouvez saisir une valeur comprise entre 0 et 120. Le routeur détecte le CA en ligne toutes les secondes. Si la valeur est de 0, cela veut dire qu'aucune détection n'est activée.

Utiliser ces serveurs DNS - Si votre FAI vous a donné l'adresse IP des serveurs DNS, cochez cette case et remplissez les champs DNS primaire et DNS secondaire. Le DNS secondaire est facultatif. Sinon, les serveurs DNS sont assignés dynamiquement par le FAI.

DNS primaire - (facultatif) Saisissez l'adresse IP DNS en notation décimale pointée fournie par votre FAI.

DNS secondaire - (facultatif) Saisissez une autre adresse IP DNS en notation décimale pointée fournie par votre FAI.

Remarque : Les nouveaux paramètres PPPoE avancés ne seront actifs que lorsque vous vous connecterez à nouveau.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.

Cliquez Précédent pour terminer.
