  • Puissance d'émission - Sélectionnez la puissance d'émission du périphérique. Vous avez le choix entre : Élevée, Moyenne ou Faible La valeur par défaut est Élevée. C'est la valeur conseillée.
  • Interv. de balise - Les balises sont des paquets envoyés par le routeur pour synchroniser un réseau sans fil. L'intervalle de balise définit le temps séparant les transmissions de signaux de balises. Vous pouvez choisir une valeur comprise entre 40 et 1000 millisecondes. La valeur par défaut est 100.
  • Seuil RTS - Permet de définir le seuil RTS (Request to Send). Si la taille du paquet est supérieure au seuil RTS indiqué, le périphérique envoie des trames RTS à une station donnée et négocie l'envoi d'une trame de données. La valeur par défaut est 2346.
  • Seuil de fragmentation - Cette valeur représente la taille maximale, conditionnant la fragmentation ou non des paquets. Une valeur trop basse peut dégrader la performance du réseau due au nombre excessif de paquets. La valeur par défaut est 2346. C'est la valeur conseillée. (Cette valeur ne peut être changée pour les séries 11N).
  • Interv. DTIM - Détermine l'intervalle de transmission entre deux messages DTIM (Delivery Traffic Indication Message). Vous pouvez choisir une valeur entre 1 et 255. La valeur par défaut est 1, ce qui signifie que l'intervalle DTIM est identique à l'intervalle de balise.
  • Activer WMM - La fonction WMM permet d'acheminer les paquets dont la priorité est élevée en premier. Il est fortement conseillé d'activer cette option.
  • Activer GI court - Fonction recommandée car elle augmente la capacité de données en réduisant l'intervalle de garde.
  • Activer isolement Point d'accès - Isole toutes les stations sans fil connectées pour éviter toute interconnexion entre elles via le WLAN. Cette fonction est désactivée si l'option WDS/Pont est active.

Remarque : Si vous n'êtes pas familier avec les paramètres de cette page, nous vous recommandons de conserver les valeurs par défaut, vous risquez sinon de diminuer les performances de votre réseau sans fil.
