Aide Journalisation système

  • Fonction Envoi automatique - Indique si la fonction Envoi automatique est activée ou non.
  • Paramètres messagerie électronique - Permet de définir l'adresse électronique de réception et d'envoi, l'adresse du serveur, les informations de validation, ainsi que l'horaire de la fonction auto-mail.
  • Type journal - Une fois le type sélectionné, seuls les journaux de ce type s'afficheront.
  • Niveau journal - Une fois le niveau sélectionné, seuls les journaux de ce niveau s'afficheront.
  • Actualiser - Rafraîchit la page pour afficher la liste des derniers journaux.
  • Enreg. le journal - Cliquez sur cette option pour enregistrer tous les journaux au format TXT.
  • Envoyer par e-mail - Cliquez sur cette option pour envoyer manuellement les journaux en cours par e-mail à l'adresse définie dans Paramètres de messagerie électronique. Le résultat s'affiche dans le journal suivant.
  • Effacer le journal - Supprime définitivement tous les journaux du périphérique.

Cliquez sur le bouton Suivant ou Précédent pour accéder à la page souhaitée.
