Aide Mise à niveau du firmware

Pour mettre à niveau le firmware du périphérique, procédez comme suit :

  1. Téléchargez la version la plus récente du fichier de mise à niveau sur notre site Web ( ).
  2. Dans le champ Nom fichier, sélectionnez ou saisissez le nom du chemin d'accès au fichier téléchargé sur le PC.
  3. Cliquez sur le bouton Mise à niveau.
  4. Le périphérique redémarrera, une fois la mise à niveau terminée.

Version firmware - Affiche la version actuelle du firmware.

Version matérielle - Affiche la version matérielle actuelle. La version matérielle du fichier de mise à niveau doit correspondre à la version matérielle utilisée.

Remarque : La version du firmware doit correspondre à celle du matériel utilisé. Le processus de mise à niveau dure quelques minutes, le périphérique redémarre automatiquement une fois cette étape terminée. Gardez l'appareil sous tension durant tout le processus. Une coupure pourrait endommager le périphérique.
