Aide Configuration réseau

Sur cette page, vous pouvez configurer les paramètres IP du réseau local.

  • Type : Sélectionnez « Smart IP(DHCP) » pour obtenir une adresse IP du serveur DHCP ou « IP statique » pour la configurer manuellement. Si vous choisissez Smart IP(DHCP), les éléments de configuration ci-dessous seront désactivés.
  • Adresse IP : Saisissez l'adresse IP de votre routeur en notation décimale pointée (adresse d'usine par défaut : ).
  • Masque de sous-réseau : code d'adresse qui détermine la taille du réseau. Le code utilisé habituellement comme masque de sous-réseau est 255.255.255.0.
  • Serveur DHCP : pour Activer ou Désactiver le serveur. Si vous désactivez le serveur, votre réseau doit disposer d'un autre serveur DHCP. Vous devrez sinon configurer manuellement l'adresse IP de l'ordinateur.

Remarque : Si vous changez l'adresse IP, vous devrez utiliser la nouvelle adresse pour vous connecter au système.
