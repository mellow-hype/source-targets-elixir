Aide Paramètres hôte

Pratique et puissante, la fonction de contrôle d'accès à Internet du routeur permet de surveiller les activités Internet des hôtes du réseau local. Sur cette page, vous pouvez définir une liste d'hôtes, nécessaire pour la règle de contrôle d'accès.

  • Description hôte - Affiche la description de l'hôte. Cette description est unique.
  • Informations - Affiche les informations sur l'hôte (IP ou MAC).
  • Modifier - Permet de modifier ou de supprimer une entrée.

Exemple : Vous souhaitez restreindre les activités Internet de l'hôte du réseau local dont l'adresse MAC est 00-11-22-33-44-AA. Vous devez d'abord procéder aux réglages suivants :

  1. Cliquez sur le bouton Ajouter... pour accéder aux Paramètres Liste d'hôtes.
  2. Dans le champ Mode, sélectionnez l'Adresse MAC dans la liste déroulante.
  3. Dans le champ Nom hôte, saisissez une description unique de l'hôte, (exemple : Hôte_1).
  4. Dans le champ Adresse MAC, saisissez 00-11-22-33-44-AA.
  5. Cliquez sur Enregistrer pour terminer.

Revenez ensuite à la page Hôte ; la liste suivante s'affiche :

ID Description hôte Informations Modifier
1 Host_1 MAC : 00-11-22-33-44-AA Modifier/Supprimer

Cliquez sur le bouton Ajouter... pour ajouter une nouvelle entrée à la liste d'hôtes.

Cliquez sur Tout Supprimer pour Supprimer toutes les entrées de la table.

Cliquez sur le bouton Suivant ou Précédent pour accéder à la page souhaitée.
