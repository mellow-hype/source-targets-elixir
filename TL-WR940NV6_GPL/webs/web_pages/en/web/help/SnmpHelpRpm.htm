Aide Paramètres SNMP

Le Simple Network Management Protocol (SNMP) est un protocole largement déployé pour surveiller et contrôler des réseaux.

Agent SNMP - Sélectionnez Activer si vous souhaitez activer la fonction de contrôle distant via l'agent SNMPv1/v2 avec MIB-II. Sélectionnez Désactiver pour fermer cette fonction.

SysContact - Identification textuelle du contact de ce nœud géré.

SysName - Nom assigné par l'administrateur au nœud géré.

SysLocation - Emplacement du nœud.

Remarque : L'utilisation de l'utilitaire en ligne pour spécifier l'une de ces valeurs active l'accès en lecture seule de l'objet. Dans le cas contraire, la requête d'écriture prime (à condition que les paramètres de contrôle d'accès le permettent), mais, au prochain redémarrage de l'agent, la valeur sera effacée.

Communauté Get - Saisissez le nom de communauté autorisant l'accès aux informations SNMP du périphérique en lecture seule. Ce nom peut être assimilé à un mot de passe. Le réglage par défaut est public.

Source Get - La source Get définit l'adresse IP ou le sous-réseau pour les systèmes de gestion qui peuvent lire les informations des périphériques de la communauté « get ».

Communauté Set - Saisissez le nom de communauté autorisant l'accès aux informations SNMP du périphérique en lecture/écriture. Ce nom peut être assimilé à un mot de passe. Le réglage par défaut est privé.

Source Set - La source Set définit l'adresse IP ou le sous-réseau pour les systèmes de gestion qui peuvent lire les informations des périphériques de la communauté « set ».

Remarque : Une source restreinte peut être une adresse IP spécifique (par exemple : 10.10.10.1) ou un sous-réseau sous forme d'IP/BITS (par exemple 10.10.10.0/24). Si une adresse IP 0.0.0.0 est spécifiée, l'agent accepte toutes les requêtes sous le nom de communauté correspondant.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.
