Aide Paramètres programmation horaire

Pratique et puissante, la fonction de contrôle d'accès à Internet du routeur permet de surveiller les activités Internet des hôtes du réseau local. Vous pouvez configurer une liste de programmations horaires afin de contrôler l'activité Internet des hôtes à un moment précis.

  • Description programmation horaire - Affiche la description de la programmation horaire. Cette description est unique.
  • Jour - Affiche le(s) jour(s) de la semaine.
  • Heure - Affiche les heures de la journée.
  • Modifier - Permet de modifier ou de supprimer une programmation horaire.

Exemple : Vous souhaitez autoriser l'hôte dont l'adresse MAC est 00-11-22-33-44-AA à accéder à www.google.com uniquement les samedis et dimanches entre 18 et 20 heures. Vous devez procéder aux réglages suivants :

  1. Cliquez sur le bouton Ajouter... pour accéder aux Paramètres Liste des programmations horaires.
  2. Dans le champ Description programmation horaire, donnez une description unique à la programmation horaire, par exemple Horaire_1.
  3. Dans le champ Jour, choisissez Sélec. jours et sélectionnez Sam et Dim.
  4. Dans le champ Heure, saisissez 1800 dans Heure de début et 2000 dans Heure de fin.
  5. Cliquez sur Enregistrer pour terminer.

Revenez ensuite à la page Programmation horaire ; la liste suivante s'affiche :

ID Description programmation horaire Jour Heure Modifier
1 Schedule_1 Sam Dim 18:00 - 20:00 Modifier/Supprimer

Cliquez sur le bouton Ajouter... pour ajouter une nouvelle entrée à la liste d'hôtes.

Cliquez sur Tout Supprimer pour Supprimer toutes les entrées de la table.

Cliquez sur le bouton Suivant ou Précédent pour accéder à la page souhaitée.
