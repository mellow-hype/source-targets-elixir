Aide UPnP

La fonction UPnP (Universal Plug and Play) permet à des périphériques (ordinateurs connectés à Internet, par exemple) d'accéder aux ressources ou périphériques du réseau local. Les périphériques UPnP sont automatiquement détectés par l'application UPnP du réseau local.

État UPnP actuel - Les boutons Activer et Désactiver permettent de (dés)activer la fonction. Cette fonction est activée par défaut.

Liste des paramètres UPnP actuels :

Cette table affiche les informations UPnP actuelles.

  • Description appli - Description de l'application qui lance la requête UPnP.
  • Port externe - Port ouvert par le routeur pour l'application.
  • Protocole - Type du protocole ouvert.
  • Port interne - Port ouvert par le routeur pour l'hôte local.
  • Adresse IP - Adresse IP de l'hôte local qui lance la requête UPnP.
  • État - Activé (le port est actif) ou Désactivé (le port est inactif).

Cliquez sur le bouton Activer pour activer la fonction UPnP.

Cliquez sur le bouton Désactiver pour désactiver la fonction UPnP.

Cliquez sur le bouton Actualiser pour rafraîchir la Liste des paramètres UPnP actuels.
