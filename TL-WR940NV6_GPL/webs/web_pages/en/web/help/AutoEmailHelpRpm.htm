Aide Paramètres compte messagerie

  • De - Votre adresse de messagerie électronique.
  • À - Adresse du destinataire.
  • Serveur SMTP - Votre serveur SMTP.
  • Authentification - La plupart des serveurs SMTP nécessitent une authentification.
  • Identifiant - Nom de votre compte de messagerie.
  • Mot de passe - Le mot de passe de votre compte de messagerie électronique.

La fonction Envoi automatique vous permet de surveiller le fonctionnement de votre périphérique.

  • Chaque jour, à une heure précise, le périphérique envoie automatiquement le journal (historique de son activité) à une adresse électronique donnée.
  • Plusieurs fois par jour, par exemple toutes les deux heures, le périphérique envoie automatiquement le journal à l'adresse électronique spécifiée.
