Aide Table ARP

Vous pouvez voir les adresses IP du réseau local et leurs adresses MAC associées en consultant la table ARP. Les boutons Charger et Supprimer vous permettent de gérer la liste.

  • Adresse MAC - Adresse MAC de l'ordinateur contrôlé du réseau local.
  • Adresse IP - Adresse IP assignée à l'ordinateur.
  • État - Indique si les adresses MAC et IP sont liées ou non.
  • Configurer - Ces boutons servent à charger ou supprimer un élément.
    • Charger - Charge l'élément de la table de liaison d'adresses IP/MAC.
    • Supprimer - Supprime l'élément de la liste.
  • Tout lier - Lie tous les éléments. Cette option est disponible uniquement lorsque la liaison ARP est activée et enregistrée sur la page Paramètres de liaison.
  • Tout charger - Charge tous les éléments de la table de liaison d'adresses IP/MAC.

Remarque : L'élément ne peut être chargé dans la table de liaison d'adresses IP/MAC si son adresse IP a déjà été chargée. Un message d'erreur s'affiche. De la même manière, « Tout charger » charge les éléments sans interférence avec la table de liaison d'adresses IP/MAC.
