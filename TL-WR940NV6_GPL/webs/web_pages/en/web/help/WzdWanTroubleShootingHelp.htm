Aide Dépannage

Les raisons les plus plausibles pour lesquelles votre routeur n'arrive pas à détecter votre type de connexion Internet sont :

  1. Il y a un problème au niveau de la connexion matérielle ou le modem n'est pas prêt.
    Afin de résoudre ce problème, vérifiez que le câble Ethernet est correctement inséré sur le port Internet (port bleu) de votre modem ou du périphérique de votre FAI (Fournisseur d'Accès Internet). Redémarrez votre modem, patientez deux minutes puis sélectionnez la première option et continuez.
  2. Vous avez une connexion Internet de type IP dynamique, mais le routeur doit cloner l'adresse MAC de l'ordinateur principal.
    Afin de résoudre ce problème, sélectionnez la deuxième option ci-dessous, puis continuez.
  3. Votre FAI (Fournisseur d'Accès Internet) vous a assigné un adresse IP fixe (statique).
    Le routeur ne peut pas détecter automatiquement un tel type de réseau. Vous devez saisir manuellement les paramètres.
