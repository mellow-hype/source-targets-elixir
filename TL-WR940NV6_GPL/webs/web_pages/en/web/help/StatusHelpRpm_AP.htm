Aide État

La page État affiche la configuration et l'état du système. Toutes ces informations sont en lecture seule.

Filaire : Informations du réseau local filaire. Vous pouvez configurer ces paramètres sur la page Réseau.

  • Adresse MAC : Adresse physique du routeur, côté LAN.
  • Adresse IP : Adresse IP LAN filaire.
  • Masque de sous-réseau : masque de sous-réseau associé à l'adresse IP.

Sans fil : Paramètres ou informations sur le réseau sans fil. Vous pouvez configurer ces paramètres sur la page Sans fil -> Paramètres Sans fil.

  • Mode de fonctionnement : Mode de fonctionnement du point d'accès.
  • Nom sans fil AP principal (distant) : SSID du point d'accès principal (distant).
  • Canal : canal sans fil utilisé.
  • Mode : Mode sans fil actuellement utilisé par le point d'accès.
  • Largeur du canal : bande passante du canal sans fil.
  • Débit d'émission max. : Débit d'émission maximale.
  • Adresse MAC : Adresse physique du point d'accès, côté WLAN.

Statistiques de trafic : Statistiques de trafic du système.

  • Envoyé(s) (octets) : Octets envoyés du port WLAN.
  • Envoyé(s) (paquets) : Paquets envoyés du port WLAN.
  • Reçu(s) (octets) : Octets reçus du port WLAN.
  • Reçu(s) (paquets) : Paquets reçus du port WLAN.

Temps disponibilité syst : Temps pendant lequel le point d'accès est disponible et en marche.

Cliquez sur le bouton Actualiser pour afficher l'état et les paramètres actuels du point d'accès.
