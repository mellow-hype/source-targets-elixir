Paramètres sans fil réseau Invité

Sur cette page, vous pouvez configurer les Paramètres sans fil du réseau Invité.

  • Autoriser invité à accéder à mon réseau local - Si cette option est activée, les invités peuvent communiquer avec les hôtes.
  • Activer Contrôle de bande passante réseau Invité - Si cette fonction est activée, les règles de contrôle de bande passante du réseau Invité s'appliquent.
  • Bande passante sortante pour réseau Invité - Débit montant passant par le port WAN pour le réseau Invité.
  • Bande passante entrante pour réseau Invité - Débit descendant passant par le port WAN pour le réseau Invité.
  • Réseau Invité (2,4 G/5 G) - Active ou désactive la fonction Réseau Invité.
  • Nom du réseau - Saisissez un nom de 32 caractères max. Tous les périphériques sans fil du réseau Invité devront utiliser ce nom (SSID).
  • Sécurité sans fil - Vous pouvez configurer la sécurité du réseau Invité sur cette page.
  • Heure d'accès - Période pendant laquelle les stations sans fil peuvent accéder au point d'accès.

Remarque :

La portée de la bande passante du réseau Invité est calculée en fonction du paramétrage de la page Contrôle de bande passante->Paramètres de contrôle.
