Aide Réseau local

Sur cette page, vous pouvez configurer les paramètres IP du réseau local.

  • Adresse MAC : adresse physique des ports LAN, côté LAN. Cette valeur ne peut être modifiée.
  • Type : Sélectionnez « Smart IP(DHCP) » pour obtenir une adresse IP du serveur DHCP ou « IP statique » pour la configurer manuellement. Si vous choisissez Smart IP(DHCP), les éléments de configuration ci-dessous seront désactivés.
  • Adresse IP : Saisissez l'adresse IP de votre routeur en notation décimale pointée (adresse d'usine par défaut : ).
  • Masque de sous-réseau : code d'adresse qui détermine la taille du réseau. Le code utilisé habituellement comme masque de sous-réseau est 255.255.255.0.
  • Passerelle : La passerelle doit appartenir au même sous-réseau que votre adresse IP.

Remarque :

  1. Si vous changez l'adresse IP, vous devrez utiliser la nouvelle adresse pour vous connecter au système.
  2. Si vous sélectionnez le type Smart IP(DHCP), le serveur DHCP sur ce périphérique ne démarrera pas lorsqu'un autre serveur DHCP se trouve au sein du réseau local.
  3. Si la nouvelle adresse IP LAN n'appartient pas au même sous-réseau, le pool d'adresses IP du serveur DHCP ne sera actif qu'une fois reconfiguré.
  4. Cliquez sur Enregistrer, le périphérique redémarre automatiquement.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.

Remarque : Avec le type Smart IP(DHCP), si votre TL-WR940N est connecté à un routeur, celui-ci enverra une Adresse IP/Passerelle dynamique à votre client. Sinon, ce dernier router obtiendra une Adresse IP/Passerelle au sein du réseau si vous définissez les réglages par défaut ci-dessous. Ce type est recommandé.
