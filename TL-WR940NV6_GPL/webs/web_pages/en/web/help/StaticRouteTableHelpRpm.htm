Aide Routage statique

Une route statique est un chemin prédéterminé que doivent emprunter les informations sur le réseau pour atteindre un hôte ou un réseau donné. Utilisez la page Routage statique pour ajouter ou supprimer une route.

Ajouter une entrée à la table de routage :

  1. Cliquez sur le bouton Ajouter....
  2. Renseignez les champs suivants :
    • Réseau de destination - Il s'agit de l'adresse IP de destination : réseau ou hôte auquel vous souhaitez assigner une route statique.
    • Masque de sous-réseau - Le masque de sous-réseau détermine quelle portion de l'adresse IP appartient au réseau et quelle autre à l'hôte.
    • Passerelle par défaut - Adresse IP de la passerelle par défaut permettant le contact entre le routeur et le réseau ou l'hôte de destination.
  3. Sélectionnez Activé dans la liste déroulante du champ État.
  4. Cliquez sur Enregistrer pour sauvegarder les modifications.

Modifier ou supprimer une entrée :

  1. Recherchez l'entrée souhaitée dans la table.
  2. Cliquez sur Modifier ou Supprimer dans la colonne Modifier.

Cliquez sur le bouton Tout activer pour activer toutes les entrées.

Cliquez sur le bouton Tout désactiver pour désactiver toutes les entrées.

Cliquez sur ce bouton Tout supprimer pour effacer toutes les entrées.
