Aide WAN IPv6

Activer IPv6 :Dés/Activation de la fonction IPv6.

Type de connexion WAN :

Sélectionnez le bon type de connexion WAN en fonction de la topologie du réseau de votre FAI.

  • SLAAC : Connexion par attribution d'une adresse IPv6 Radvd.
  • DHCPv6 : Connexion par attribution d'une adresse IPv6 dynamique.
  • IPv6 statique : Connexion par attribution d'une adresse IPv6 statique.
  • PPPoEv6 : Connexion PPPoEv6 nécessitant un identifiant et un mot de passe.
  • Tunnel 6to4 : Connexion par attribution d'une adresse 6to4.

Adresse  : Adresse IPv4 assignée dynamiquement par votre FAI.

Masque de sous-réseau : Adresse IPv4 assignée dynamiquement par votre FAI.

Passerelle par défaut : Passerelle IPv4 par défaut assignée dynamiquement par votre FAI.

Adresse tunnel : Adresse tunnel 6to4 créée par le périphérique pour accéder au réseau IPv6.

Taille MTU : La Valeur de la MTU (Maximum Transmit Unit, unité de transmission maximale ) normale pour la plupart des réseaux Ethernet est 1500 octets. Cependant, en fonction de votre FAI, il se peut que vous deviez la changer. Toutefois, cela est rarement nécessaire ; ne procédez à la modification que si cela est requis pour votre connexion FAI.

Si votre FAI vous a fourni une ou deux adresses IPv6 DNS, sélectionnez l'option Utiliser les serveurs DNS IPv6 suivants et remplissez les champs DNS IPv6 primaire et DNS IPv6 secondaire. Sinon, les serveurs DNS sont assignés dynamiquement par le FAI.

DNS IPv6 primaire : saisissez l'adresse IPv6 DNS en notation hexadécimale séparée par des doubles points fournie par votre FAI.

DNS IPv6 secondaire : saisissez une autre adresse IPv6 DNS en notation décimale séparée par des doubles points fournie par votre FAI.

Remarque : Si le message d'erreur « Adresse introuvable » s'affiche lorsque vous accédez à un site Internet, cela peut provenir d'un mauvais paramétrage des serveurs DNS. Nous vous conseillons dans ce cas de contacter votre FAI pour obtenir les adresses correctes.

Configuration LAN

Type d'autoconfiguration d'adresse : Serveur RADVD (Router Advertisement Daemon) et DHCPv6 (Dynamic Host Configuration Protocol for IPv6).

Type de configuration de préfixe de site : Type de préfixe d'adresse IPv6.

  • Délégué : Obtention automatique du préfixe d'adresse IPv6 auprès du FAI ; le périphérique le délègue au réseau LAN.
  • Statique : Configuration manuelle du Préfixe du site et de la Longueur du préfixe de site. Veuillez contacter votre FAI pour en savoir plus sur leur configuration.

Bail : Cette option définit la durée pendant laquelle un utilisateur réseau est autorisé à se connecter au routeur avec l'adresse DHCPv6 en cours. Saisissez la durée du bail de l'adresse DHCPv6 en secondes. L'intervalle va de 1 à 691 200 secondes. La valeur par défaut est de 86 400 secondes.

Adresse IPv6 LAN : affiche l'adresse vIPv6 LAN. créé par le périphérique.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.
