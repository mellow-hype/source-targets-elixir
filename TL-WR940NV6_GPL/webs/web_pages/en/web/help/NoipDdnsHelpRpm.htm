Aide DDNS

Le routeur offre une fonction DDNS (système de noms de domaine dynamique). Le DDNS vous permet d'assigner un nom de domaine et d'hôte fixes à une adresse IP dynamique, option utile si vous hébergez votre site Web, serveur FTP ou tout autre serveur derrière le routeur. Avant d'utiliser cette fonction, vous devez vous inscrire sur le site de fournisseurs de service DDNS, tels que www.noip.com. Le fournisseur de service DDNS vous fournira un mot de passe ou une clé.

Afin de configurer le DDNS, procédez comme suit :

Si votre Fournisseur de service DDNS est www.noip.com,

  1. Saisissez l'Identifiant de votre compte DDNS.
  2. Saisissez le Mot de passe de votre compte DDNS.
  3. Saisissez le Nom de domaine fourni par votre fournisseur de service DDNS.
  4. Cliquez sur le bouton Identification pour vous connecter au service DDNS.

État de la connexion - Affiche l'état de la connexion au service DDNS.

Cliquez sur Déconnexion pour quitter le service DDNS.

Note :  Si vous souhaitez vous connecter avec un autre compte, cliquez sur le bouton Déconnexion, puis saisissez les identifiant et mot de passe correspondants et cliquez sur Connexion.
