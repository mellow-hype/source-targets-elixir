Aide Mot de passe

Il est fortement conseillé de changer l'identifiant et le mot de passe par défaut de ce périphérique. Tous les utilisateurs qui tentent d'accéder à l'utilitaire Web de ce périphérique se verront demander leurs identifiant et mot de passe.

Remarque : Les nouveaux identifiant et mot de passe ne doivent pas dépasser 14 caractères, ni inclure d'espace. Saisissez le nouveau mot de passe deux fois pour le confirmer.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.

Cliquez sur Tout effacer pour tout supprimer.
