Aide Gestion des règles de contrôle d'accès à Internet

Pratique et puissante, la fonction de contrôle d'accès à Internet du routeur permet de surveiller les activités Internet des hôtes du LAN. En outre, vous pouvez combiner les Liste d'hôtes, Liste de cibles et Programmation horaire pour restreindre la navigation Internet des hôtes.

  • Activer le contrôle d'accès à Internet - Cochez la case pour activer cette fonction et activer la règle de filtrage par défaut.
  • Nom de règle - Affiche le nom de la règle, ce nom est unique.
  • Hôte - Affiche l'hôte sélectionné dans la règle correspondante.
  • Cible - Affiche la cible sélectionnée dans la règle correspondante.
  • Programmation horaire - Affiche la programmation horaire sélectionnée dans la règle correspondante.
  • Activer - Cochez cette case pour activer une entrée spécifique.
  • Modifier - Permet de modifier ou supprimer une règle existante.

Exemple : Vous souhaitez autoriser l'hôte dont l'adresse MAC est 00-11-22-33-44-AA à accéder à www.google.com uniquement les samedis et dimanches entre 18 et 20 heures et interdire aux autres hôtes du réseau local d'accéder à Internet. Vous devez procéder aux réglages suivants :

  1. À gauche de l'écran, dans Contrôle d'accès, cliquez sur le sous-menu Règle pour revenir à la page Liste de règles. Sélectionnez Activer le contrôle d'accès à Internet, puis « Autoriser les paquets spécifiés par une politique de contrôle d'accès activée à traverser le routeur ».
  2. Nous vous conseillons de cliquer sur le bouton Assistant d'installation pour appliquer les paramètres suivants.
  3. À gauche de l'écran, dans Contrôle d'accès, cliquez sur le sous-menu Hôte pour accéder à la page Liste d'hôtes. Ajoutez une nouvelle entrée avec comme description de l'hôte : Hôte_1 et l'adresse MAC : 00-11-22-33-44-AA.
  4. À gauche de l'écran, dans Contrôle d'accès, cliquez sur le sous-menu Cible afin d'accéder à la Liste de cibles. Ajoutez une nouvelle entrée avec comme description de la cible : Cible_1 et le nom de domaine : www.google.com.
  5. À gauche de l'écran, dans Contrôle d'accès, cliquez sur le sous-menu Programmation horaire pour accéder à la Liste des programmations horaires. Ajoutez une nouvelle entrée avec comme description : Programmation_1, Jour : Sam et Dim, Heure de début : 1800 et Heure de fin : 2000.
  6. À gauche de l'écran, dans Contrôle d'accès, cliquez sur le sous-menu Règle , puis sur le bouton Ajouter... pour créer une nouvelle règle comme suit :
 
  • Dans le champ Nom de règle, donnez un nom à la règle. Important : Ce nom doit être unique (ex. : Règle_1).
  • Dans le champ Hôte, sélectionnez Hôte_1.
  • Dans le champ Cible, sélectionnez Cible_1.
  • Dans le champ Programmation horaire, sélectionnez Horraire_1.
  • Dans le champ État, sélectionnez Activer.
  • Cliquez sur Enregistrer pour terminer.

Revenez ensuite à la page Gestion des règles de contrôle d'accès ; la liste suivante s'affiche :

ID Règle Hôte Cible Programmation horaire État Modifier
1 Rule_1 Host_1 Target_1 Schedule_1 Modifier/Supprimer

Cliquez sur le bouton Assistant d'installation pour créer une règle.

Cliquez sur le bouton Ajouter... pour ajouter une nouvelle règle.

Cliquez sur le bouton Tout activer pour activer toutes les règles de la liste.

Cliquez sur le bouton Tout désactiver pour désactiver toutes les règles de la liste.

Cliquez sur Tout Supprimer pour Supprimer toutes les entrées de la table.

Cliquez sur le bouton Suivant ou Précédent pour accéder à la page souhaitée.
