Aide Paramètres de configuration du réseau sans fil

Remarque : La portée de votre connexion sans fil peut grandement varier selon la position de votre point d'accès. Pour des résultats optimaux, il est important de placer votre point d'accès :

  • au centre de vos stations sans fil.
  • en hauteur, par exemple sur une étagère.
  • loin de toute source d'interférences potentielle : ordinateurs, four à micro-ondes, téléphones sans fil...
  • avec l'antenne en position verticale.
  • loin de grandes surfaces métalliques.

Point d'accès : mode de fonctionnement pour l'accès des stations sans fil.

  • Nom du réseau sans fil : saisissez un nom de 32 caractères max. Tous les périphériques sans fil du réseau devront utiliser ce même nom (Nom du réseau sans fil).
  • Canal : indique la fréquence de fonctionnement. Il n'est pas nécessaire de changer le canal sans fil, sauf en cas d'interférences constatées avec un autre point d'accès à proximité.
  • Mode : ce champ indique le mode de fonctionnement sans fil du point d'accès.
  • Largeur du canal : bande passante du canal sans fil.
  • Activer radio sans fil : la radio sans fil du point d'accès peut être (dés)activée afin d'autoriser ou de refuser l'accès des stations sans fil. Activé : les stations sans fil peuvent accéder au point d'accès. Désactivé : les stations ne peuvent accéder au point d'accès.
  • Activer diffusion SSID : cochez l'option Activer la diffusion SSID si vous souhaitez que le point d'accès diffuse son nom (SSSID).

Remarque : Le non-respect de ces consignes peut entraîner une forte dégradation des performances, voire l'impossibilité d'activer la connexion sans fil du point d'accès.

Répéteur : En mode Répéteur, le point d'accès avec WDS désactivé transmet les données à un point d'accès principal associé. La fonction Point d'accès est activée. Le répéteur sans fil relaie le signal entre ses stations et le point d'accès principal pour augmenter la portée du sans fil. Veuillez saisir l'adresse MAC du point d'accès principal dans le champ Adresse MAC du point d'accès principal.

  • Nom sans fil du point d'accès principal : SSID du point d'accès auquel va se connecter votre périphérique en tant que client. Vous pouvez également utiliser la fonction de recherche pour sélectionner le SSID.
  • Adresse MAC du point d'accès principal : BSSID du point d'accès auquel va se connecter votre périphérique, en tant que client. Vous pouvez également utiliser la fonction de recherche pour sélectionner le BSSID.
  • Mode -Indique le mode de fonctionnement sans fil du point d'accès.
  • Largeur du canal : bande passante du canal sans fil.
  • Mode WDS : Ce champ indique le mode WDS utilisé. Inutile de le changer, sauf en cas de problèmes de communication réseau avec le point d'accès principal. En mode automatique, le routeur choisit automatiquement le mode WDS adéquat.
  • Activer radio sans fil : La radio sans fil du point d'accès peut être (dés)activée afin d'autoriser ou de refuser l'accès des stations sans fil. Activé : les stations sans fil peuvent accéder au point d'accès. Désactivé : les stations ne peuvent accéder au point d'accès.
  • Liste : Cliquez sur ce bouton pour rechercher le point d'accès qui utilise le canal.
