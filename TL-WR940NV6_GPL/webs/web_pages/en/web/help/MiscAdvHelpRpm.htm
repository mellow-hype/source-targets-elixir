Aide Liste des hôtes DoS bloqués

Cette page affiche l'Adresse IP hôte et l'Adresse MAC hôte des hôtes bloqués par le routeur.

Pour rafraîchir la page et afficher la liste des hôtes bloqués, cliquez sur le bouton Actualiser.

Cliquez sur le bouton Effacer pour supprimer toutes les entrées.

Cliquez sur le bouton Précédent pour revenir à la page Sécurité Avancée.
