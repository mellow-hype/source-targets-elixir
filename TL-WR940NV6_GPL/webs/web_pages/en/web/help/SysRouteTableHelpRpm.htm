Aide Table de routage système

La table de routage système contient toutes les routes valides en cours d'utilisation. Pour chaque entrée, sont indiqués l'adresse IP de destination, le masque de sous-réseau, la passerelle et l'interface. Cliquez sur le bouton Actualiser pour rafraîchir les données.

  • Réseau de destination - Adresse du réseau ou de l'hôte auquel vous souhaitez assigner une route statique
  • Masque de sous-réseau - Le masque de sous-réseau détermine quelle portion de l'adresse IP appartient au réseau et quelle autre à l'hôte.
  • Passerelle - Adresse IP de la passerelle permettant le contact entre le routeur et le réseau ou l'hôte de destination.
  • Interface - Cette interface vous indique où se situe l'adresse IP de destination : LAN/WLAN (réseaux locaux filaire et sans fil) ou WAN (Internet).
