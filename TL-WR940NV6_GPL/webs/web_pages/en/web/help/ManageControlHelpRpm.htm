Aide Gestion distante

Cette fonction vous permet de gérer votre routeur à distance via Internet.

  • Port de gestion distante - Le navigateur Web utilise normalement le port de service HTTP standard. Par défaut, le port de gestion distante du routeur est 80. Pour plus de sécurité, vous pouvez personnaliser le port en entrant le numéro de votre choix dans la case prévue à cet effet. Choisissez un numéro compris entre 1 et 65535, mais n'utilisez pas de numéro de port de service standard.
  • Adresse IP Gestion distante - Il s'agit de l'adresse utilisée lorsque vous accédez au routeur via Internet. Cette fonction est désactivée si l'adresse IP est réglée sur 0.0.0.0. Pour l'activer, remplacez cette valeur par défaut par une adresse IP valide. Si l'adresse IP est définie sur 255.255.255.255, tous les hôtes peuvent accéder au routeur via Internet.

Pour accéder au routeur, saisissez son adresse IP WAN dans la barre d'adresse de votre navigateur Web, suivie de deux points et du numéro de port que vous avez entré dans la case Port de gestion distante. Ainsi, si l'adresse WAN de votre routeur est 202.96.12.8 et le numéro de port est 8080, saisissez http://202.96.12.8:8080 dans votre navigateur. Le mode de passe du routeur vous est demandé. Vous ne pourrez accéder à l'utilitaire en ligne du routeur qu'une fois le mot de passe saisi avec succès.

Remarque :

  1. Certains ports sont habituellement utilisés pour d'autres services (tels que les 21, 25, 110, 119, 139, 145 et 445). Pour des raisons de sécurité, ces ports seront restreints.
  2. Veillez à changer le mot de passe par défaut du routeur pour un mot de passe sécurisé.
  3. Si le port de gestion distante est en conflit avec celui du Serveur Virtuel, l'entrée sera automatiquement désactivée une fois les paramètres enregistrés.
