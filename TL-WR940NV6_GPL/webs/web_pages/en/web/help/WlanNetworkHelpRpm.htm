Aide Paramètres de configuration du réseau sans fil

Remarque : La portée de votre connexion sans fil peut grandement varier selon la position de votre routeur. Pour des résultats optimaux, il est important de placer votre routeur :

  • au centre de vos stations sans fil.
  • en hauteur, par exemple sur une étagère.
  • loin de toute source d'interférences potentielle : ordinateurs, four à micro-ondes, téléphones sans fil...
  • avec l'antenne en position verticale.
  • loin de grandes surfaces métalliques.

Remarque : Le non-respect de ces consignes peut entraîner une forte dégradation des performances, voire l'impossibilité d'activer la connexion sans fil au routeur.

Nom du réseau sans fil - Saisissez un nom de 32 caractères max. Tous les périphériques sans fil du réseau devront utiliser ce nom (SSID).

Région - Sélectionnez votre région dans la liste déroulante. Ce champ liste les régions où il est possible d'utiliser la fonction sans fil du routeur. Attention : Utiliser le routeur en mode sans fil dans une région non listée peut être illégal. Si votre pays ou région n'apparaît pas dans la liste, veuillez contacter l'administration locale compétente.

Mode - Sélectionnez le mode de transmission en fonction de votre appareil sans fil.

Largeur du canal - Bande passante du canal sans fil.

Canal - Indique la fréquence de fonctionnement. Il n'est pas nécessaire de changer le canal sans fil, sauf en cas d'interférences constatées avec un autre point d'accès à proximité. En mode automatique, le point d'accès choisit automatiquement le meilleur canal.

Activer radio routeur sans fil - La radio sans fil du routeur peut être (dés)activée pour autoriser ou non l'accès aux stations sans fil. Activé : les stations peuvent accéder au routeur. Désactivé : les stations ne peuvent accéder au routeur.

Activer diffusion SSID - Cochez cette option Activer la diffusion SSID si vous souhaitez que le routeur sans fil diffuse son nom (SSSID).

Activer pontage WDS - Cette fonction permet au routeur d'établir un pont entre plusieurs réseaux locaux sans fil. REMARQUE : Si cette case est cochée, assurez-vous que les paramètres suivants sont corrects :

SSID (à ponter) - SSID du point d'accès auquel va se connecter votre routeur, en tant que client. Utilisez également la fonction Liste pour sélectionner le SSID.

BSSID (à ponter) - BSSID du point d'accès auquel va se connecter votre routeur? en tant que client. Utilisez également la fonction Liste pour sélectionner le BSSID.

Liste - Cliquez sur ce bouton pour rechercher le point d'accès qui utilise le canal.

Mode WDS - Ce champ indique le mode WDS utilisé. Inutile de le changer, sauf en cas de problèmes de communication réseau avec le point d'accès principal. En mode automatique, le routeur choisit automatiquement le mode WDS adéquat.

Type clé - Cette option doit être choisie en fonction de la configuration de sécurité du point d'accès. Le type de sécurité doit être le même que celui du point d'accès.

Index WEP - Cette option doit être choisie si le type de clé est WEP (ASCII) ou WEP (HEX). Elle indique l'index de la clé WEP.

Type d'auth. - Cette option doit être choisie si le type de clé est WEP (ASCII) ou WEP (HEX). Elle indique le type d'autorisation du point d'accès principal.

Mot de passe - Si le point d'accès auquel votre routeur va se connecter requiert un mot de passe, saisissez-le dans ce champ.
