Aide Paramètres de liaison

La liaison ARP est utile pour contrôler l'accès de certains ordinateurs du réseau local.

  • Adresse MAC - Adresse MAC de l'ordinateur contrôlé.
  • Adresse IP - Adresse IP assignée à l'ordinateur.
  • Lier - Cochez cette option pour activer la liaison ARP d'un périphérique donné.
  • Modifier -Permet de modifier ou de supprimer une entrée.
  • Ajouter... - Cliquez sur le bouton Ajouter... pour créer une entrée.
  • Tout activer - Cliquez sur ce bouton pour activer toutes les entrées.
  • Tout désactiver - Cliquez sur ce bouton pour désactiver toutes les entrées.
  • Tout supprimer - Cliquez sur le bouton Tout supprimer pour effacer toutes les entrées.
  • Rechercher - Permet de retrouver une entrée.

    Exemple : Vous souhaitez utiliser la liaison pour assigner l'adresse IP 192.168.0.23 au PC A (MAC : 00-E0-4C-00-07-BE) et empêcher les autres ordinateurs d'utiliser cette adresse. Activez d'abord la « Liaison ARP », puis ajoutez un nouvel élément dans la table de liaison afin qu'elle soit semblable à la table ci-dessous.
