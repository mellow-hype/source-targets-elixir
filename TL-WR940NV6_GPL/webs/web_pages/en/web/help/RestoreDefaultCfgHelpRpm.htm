Aide Réglages d'usine

Cliquez sur Restaurer pour réinitialiser tous les paramètres de configuration par défaut.

  • Identifiant par défaut - admin.
  • Mot de passe par défaut - admin.
  • Adresse IP par défaut - 192.168.0.1.
  • Masque de sous-réseau par défaut - 255.255.255.0.

Remarque : Une fois les réglages restaurés, tous vos changements de paramètres seront perdus.
