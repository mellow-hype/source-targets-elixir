Aide PPPoE

Identifiant et Mot de passe - Saisissez l'identifiant et le mot de passe fournis par votre FAI. Attention, ces champs sont sensibles à la casse.

Connexion secondaire - Connexion secondaire - Disponible uniquement pour la connexion PPoE. Si votre FAI fournit un type de connexion supplémentaire (IP dynamique/statique) pour se connecter à un réseau local, vous pouvez cocher le bouton radio IP dynamique/statique pour activer cette connexion secondaire.

  • Désactivé - La connexion secondaire est par défaut désactivée. Vous ne disposez donc que d'une connexion PPoE. Ce paramétrage est recommandé.
  • IP dynamique - Utilisez l'adresse IP dynamique pour vous connecter au réseau local fourni par votre FAI.
  • IP statique - Utilisez l'adresse IP statique pour vous connecter au réseau local fourni par votre FAI. Adresse IP - Adresse IP - Saisissez l'adresse IP fournie par votre FAI pour la connexion secondaire. Cette adresse sert uniquement pour l'accès au réseau local de la connexion secondaire. Masque de sous-réseau - Saisissez le masque de sous-réseau fourni par votre FAI pour la connexion secondaire.
