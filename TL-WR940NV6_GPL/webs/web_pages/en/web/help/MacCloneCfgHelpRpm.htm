Aide Clonage MAC

Certains FAI vous demandent d'enregistrer l'adresse MAC de votre adaptateur. Normalement, vous n'avez rien à modifier.

Adresse MAC WAN - Ce champ affiche l'adresse MAC en cours du port WAN. Si votre FAI vous le demande, saisissez l'adresse MAC à enregistrer dans ce champ. Le format de l'adresse MAC est XX-XX-XX-XX-XX-XX (X = tout chiffre hexadécimal).

Adresse MAC du PC - Ce champ affiche l'adresse MAC de l'ordinateur qui gère le routeur. Si vous devez reporter cette adresse MAC, utilisez le bouton Cloner l'adresse MAC pour la copier dans le champ Adresse MAC WAN.

Cliquez sur Restaurer MAC par défaut pour restaurer les paramètres par défaut de l'adresse MAC du port WAN.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.

Remarque :

  1. Seuls les ordinateurs de votre réseau local peuvent utiliser cette fonctionnalité.
  2. Si vous changez l'Adresse MAC WAN avec une connexion WAN de type PPPoE, cette modification sera active une fois la connexion rétablie.
