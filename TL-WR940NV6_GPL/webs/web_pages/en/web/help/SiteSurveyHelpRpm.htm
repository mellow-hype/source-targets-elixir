Aide Liste des réseaux WLAN

Remarque : Les informations sur les points d'accès que vous pouvez connecter s'affichent sur cette page. Procédez comme suit :

  • Tout d'abord, recherchez la ligne sur laquelle se trouve le réseau auquel vous souhaitez vous connecter.
  • Puis, cliquez sur le lien hypertexte « Connecter » en fin de ligne.
  • Le réseau SSID cible est saisi automatiquement au bon endroit dans la page de configuration du réseau local sans fil.

Remarque : Cliquez sur Actualiser pour mettre à jour la liste des points d'accès.

Cliquez sur Précédent pour retourner sur la page de configuration du réseau local sans fil.
