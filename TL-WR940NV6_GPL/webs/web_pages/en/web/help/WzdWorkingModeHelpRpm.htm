Aide Mode de fonctionnement

Routeur sans fil standard : Avec ce mode, le périphérique n'utilise que le port WAN pour accéder à Internet. Les hôtes internes peuvent accéder à Internet via quatre ports LAN ou en mode sans fil.

Point d'accès : Dans ce mode, le périphérique peut être connecté à un réseau filaire et transformer l'accès filaire en sans fil qui sera partagé entre plusieurs périphériques. Ce mode est recommandé pour votre domicile, un bureau ou un hôtel où seul un réseau filaire est disponible.

Range Extendeur : Dans ce mode, le périphérique peut copier et renforcer le signal sans fil existant afin d'étendre sa couverture. Ce mode est particulièrement pratique dans un grand espace, afin de réduire les « zones mortes ».
