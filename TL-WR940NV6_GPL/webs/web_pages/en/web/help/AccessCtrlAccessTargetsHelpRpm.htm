Aide Paramètres cible

Pratique et puissante, la fonction de contrôle d'accès à Internet du routeur permet de surveiller les activités Internet des hôtes du réseau local. Sur cette page, vous pouvez créer une liste de cibles pour restreindre ou non l'accès des hôtes.

  • Description cible - Affiche la description de la cible. Cette description est unique.
  • Informations - La cible peut être une adresse IP, un port ou un nom de domaine.
  • Modifier - Permet de modifier ou de supprimer une entrée.

Exemple : Vous souhaitez restreindre les activités Internet de l'hôte du réseau local dont l'adresse MAC est 00-11-22-33-44-AA afin qu'il n'accède qu'à www.google.com. Vous devez procéder aux réglages suivants :

  1. Cliquez sur le bouton Ajouter... pour accéder aux Paramètres Liste de cibles.
  2. Dans le champ Mode, sélectionnez un nom de domaine dans la liste déroulante.
  3. Dans le champ Description cible, saisissez une description unique, (exemple : Cible_1).
  4. Dans le champ Nom de domaine, saisissez www.google.com.
  5. Cliquez sur Enregistrer pour terminer.

Revenez ensuite à la page Cible ; la liste suivante s'affiche :

ID Description cible Informations Modifier
1 Target_1 www.google.com Modifier/Supprimer

Cliquez sur le bouton Ajouter... pour ajouter une nouvelle cible.

Cliquez sur le bouton Tout supprimer pour effacer toutes les entrées de la table.

Cliquez sur le bouton Suivant ou Précédent pour accéder à la page souhaitée.
