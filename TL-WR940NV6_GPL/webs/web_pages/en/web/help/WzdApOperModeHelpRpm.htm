Mode de fonctionnement - Plusieurs modes de fonctionnement sont pris en charge : (1) Point d'accès (2) Répéteur (3) Pont avec PA (4) Client (5) Multi-SSID. Voici une présentation des options de configuration disponibles, diffèrent en fonction du mode choisi.

Point d'accès - Dans ce mode, le périphérique peut être connecté à un réseau filaire et transformer l'accès filaire en sans fil qui sera partagé entre plusieurs périphériques. Ce mode est recommandé pour votre domicile, un bureau ou un hôtel où seul un réseau filaire est disponible.

Répéteur - Dans ce mode, le périphérique peut copier et renforcer le signal sans fil existant afin d'étendre sa couverture. Ce mode est particulièrement pratique dans un grand espace, afin de réduire les « zones mortes ».

Pont avec point d'accès - Dans ce mode, le périphérique peut servir à combiner plusieurs réseaux locaux trop éloignés les uns des autres, afin de n'en créer qu'un seul, accessible par connexion sans fil. Tout en établissant un pont avec les autres points d'accès, ce périphérique peut aussi fonctionner en tant que point d'accès pour créer un réseau local sans fil pour tous vos périphériques Wi-Fi.

Client - Dans ce mode, le périphérique peut être connecté à un autre périphérique via un port Ethernet et fonctionner comme un adaptateur afin d'offrir aux périphériques filaires un accès au réseau sans fil. Ce mode est particulièrement conseillé pour une télévision connectée, un lecteur média ou une console de jeu disposant d'un port Ethernet.

Multi-SSID - Dans ce mode, le périphérique peut créer jusqu'à quatre réseaux sans fil identifiés par des SSID différents et assigner chacun d'eux à un VLAN ou un type de sécurité différent. Ce mode est particulièrement conseillé lorsque différentes politiques d'accès et fonctions sont requises.

Remarque : La portée de votre connexion sans fil peut grandement varier selon la position de votre point d'accès. Pour des résultats optimaux, il est important de placer votre point d'accès :

  • au centre de vos stations sans fil.
  • en hauteur, par exemple sur une étagère.
  • loin de toute source d'interférences potentielle : ordinateurs, four à micro-ondes, téléphones sans fil...
  • avec l'antenne en position verticale.
  • loin de grandes surfaces métalliques.
