Aide WAN

Type de connexion WAN :

Si votre FAI utilise un serveur DHCP, sélectionnez l'option IP dynamique.

Si votre FAI fournit une adresse IP statique ou fixe, un masque de sous-réseau, une passerelle et des paramètres DNS, sélectionnez l'option IP statique.

Si votre FAI fournit une connexion PPPoE, sélectionnez l'option PPPoE/Russia PPPoE.

Si votre FAI fournit une connexion par câble BigPond, sélectionnez l'option Câble BigPond.

Si votre FAI fournit une connexion L2TP, sélectionnez l'option L2TP/Russia L2TP.

Si votre FAI fournit une connexion PPTP, sélectionnez l'option PPTP/Russia PPTP.

Identifiant/Mot de passe - Saisissez l'identifiant et le mot de passe fournis par votre FAI. Attention, ces champs sont sensibles à la casse.

IP dynamique/IP statique - Si votre FAI fournit une adresse IP, un masque de sous-réseau, une passerelle et une adresse de serveur DNS, sélectionnez IP statique. Sinon, sélectionnez IP dynamique.

Adresse IP/Nom du serveur - Saisissez l'adresse IP ou le nom de domaine du serveur fourni par votre FAI.

Adresse IP - Entrez l'adresse IP utilisée pour l'accès au réseau. (Configurable si l'option IP statique est sélectionnée)

Masque de sous-réseau - Saisissez le masque de sous-réseau fourni par votre FAI. (Configurable si l'option IP statique est sélectionnée)

Passerelle - Saisissez la passerelle fournie par votre FAI. (Configurable si l'option IP statique est sélectionnée)

DNS - Saisissez le serveur DNS fourni par votre FAI. (Configurable si l'option IP statique est sélectionnée)

Adresse IP Internet Adresse IP Internet assignée par le serveur PPTP.

DNS Internet Adresse du serveur DNS Internet assignée par le serveur PPTP.

Connexion à la demande - Vous pouvez configurer le routeur pour qu'il se déconnecte d'Internet après une période spécifiée (Délai d'inactivité max.). Si votre session a expiré pour cause d'inactivité, l'option Connexion à la demande permet au routeur de rétablir automatiquement la connexion dès que vous tentez d'accéder à nouveau à Internet. Pour activer cette option, cliquez sur le bouton radio. Pour que votre connexion Internet soit active en permanence, entrez 0 dans le champ Délai d'inactivité max.. Sinon, saisissez le nombre de minutes que vous souhaitez voir s'écouler avant l'interruption de votre connexion Internet.

Avertissement - Parfois la connexion ne peut être interrompue, même en ayant spécifié un délai d'inactivité max., car certaines applications fonctionnent en arrière-plan.

Connexion automatique - Lance la connexion automatiquement après déconnexion du routeur. Pour activer cette option, cliquez sur le bouton radio.

Connexion manuelle - Vous pouvez configurer le routeur afin de (dés)activer manuellement la connexion. Après une période d'inactivité donnée (Délai d'inactivité max.), le routeur suspend la connexion Internet. Si vous tentez d'y accéder à nouveau, le routeur ne peut pas la rétablir automatiquement. Pour activer cette option, cliquez sur le bouton radio. Pour que votre connexion Internet soit active en permanence, entrez 0 dans le champ Délai d'inactivité max.. Sinon, saisissez le nombre de minutes que vous souhaitez voir s'écouler avant que votre connexion Internet soit interrompue, à moins qu'une nouvelle liaison soit nécessaire.

Avertissement - Parfois la connexion ne peut être interrompue, même en ayant spécifié un délai d'inactivité max., car certaines applications fonctionnent en arrière-plan.

Cliquez sur le bouton Connexion pour vous connecter immédiatement.

Cliquez sur le bouton Déconnexion pour vous déconnecter immédiatement.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.
