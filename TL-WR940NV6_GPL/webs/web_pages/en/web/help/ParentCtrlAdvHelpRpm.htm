Aide Ajouter ou modifier un contrôle parental

Grâce à la fonction pratique de contrôle parental, le routeur peut restreindre l'accès à certains sites Web et le temps de navigation de votre enfant sur Internet. Sur cette page, vous pouvez créer une règle.

  • Adresse MAC du PC enfant - Saisissez l'adresse MAC de l'ordinateur que vous souhaitez contrôler ou utiliser l'option Toutes les adresses MAC du réseau local ci-dessous. Si vous laissez le champ vide, la règle s'appliquera à tous les ordinateurs, à l'exception du PC parental.
  • Toutes les adresses MAC du réseau local - Pour consulter les adresses MAC de tous les ordinateurs du réseau local en cours, cliquez sur le menu déroulant. Sélectionnez-en-une ; elle sera ensuite copiée dans le champ Adresse MAC du PC de l'enfant.
  • Description site Web - Dans ce champ, décrivez le(s) site(s) Web. Important : Cette description doit être unique.
  • Nom des sites Web autorisés - Vous pouvez saisir huit noms de domaine auxquels l'enfant peut accéder, en indiquant soit le nom complet, soit des mots clés (par exemple : google). Tout nom de domaine comportant ces mots clés (www.google.com, news.google.com) est autorisé.
  • Horaires effectifs - Sélectionnez les horaires effectifs de cette règle ou utilisez l'option Contrôle d'accès > Programmation horaire. La valeur par défaut est Aucune.
  • État - Deux options sont disponibles : Activer ou Désactiver. La première applique la règle, la seconde ne l'applique pas.

Cliquez sur Enregistrer pour terminer.

Cliquez sur Précédent pour revenir aux Paramètres de contrôle parental.
