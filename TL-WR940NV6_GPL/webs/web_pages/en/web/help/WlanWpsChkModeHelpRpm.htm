Aide Ajouter un périphérique

Cette section vous permet d'ajouter un nouveau périphérique sans fil à un réseau existant. Si le périphérique est compatible WPS (Wi-Fi Protected Setup) et doté d'un bouton de configuration, il vous suffit d'appuyer dessus, puis de sélectionner Appuyer sur le bouton du nouveau périphérique dans deux minutes. Cliquez ensuite dans les deux minutes sur le bouton Connexion de cette page de configuration en ligne. Pour ajouter un nouveau périphérique, vous pouvez aussi saisir son code PIN et cliquer sur le bouton Connexion.

  • Saisir le code PIN du nouveau périphérique - Sélectionnez ce paramètre si le nouveau périphérique prend en charge ce moyen de connexion.
  • PIN - Saisissez le code PIN dans ce champ.
  • Appuyer sur le bouton du nouveau périphérique dans deux minutes - Sélectionnez ce paramètre si le nouveau périphérique prend en charge ce moyen de connexion.
  • Précédent - Permet de revenir à la page de configuration.
  • Connexion - Permet de connecter le nouveau périphérique et de l'ajouter au réseau.
