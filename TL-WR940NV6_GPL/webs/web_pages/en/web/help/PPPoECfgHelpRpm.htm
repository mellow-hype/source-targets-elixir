Aide WAN

Type de connexion WAN :

Si votre FAI utilise un serveur DHCP, sélectionnez l'option IP dynamique.

Si votre FAI fournit une adresse IP statique ou fixe, un masque de sous-réseau, une passerelle et des paramètres DNS, sélectionnez l'option IP statique.

Si votre FAI fournit une connexion PPPoE, sélectionnez l'option PPPoE/Russia PPPoE.

Si votre FAI fournit une connexion par câble BigPond, sélectionnez l'option Câble BigPond.

Si votre FAI fournit une connexion L2TP, sélectionnez l'option L2TP/Russia L2TP.

Si votre FAI fournit une connexion PPTP, sélectionnez l'option PPTP/Russia PPTP.

Remarque : Si vous ne savez pas quel type de connexion choisir, cliquez sur le bouton Détecter pour que le routeur détermine automatiquement la connexion Internet pour les serveurs et les protocoles. Le type de connexion s'affiche dès que le routeur détecte une connexion Internet active. Cette information est donnée à titre indicatif uniquement. Pour vous assurer du type de connexion fournie, consultez votre FAI. Le routeur peut détecter les types de connexion Internet suivants :

  • PPPoE/Russia PPPoE - Connexion PPPoE nécessitant un identifiant et un mot de passe.
  • IP dynamique - Connexion par attribution d'une adresse IP dynamique.
  • IP statique - Connexion par attribution d'une adresse IP statique.

Identifiant/Mot de passe - Saisissez l'identifiant et le mot de passe fournis par votre FAI. Attention, ces champs sont sensibles à la casse.

Connexion secondaire - Connexion secondaire - Disponible uniquement pour la connexion PPoE. Si votre FAI fournit un type de connexion supplémentaire (IP dynamique/statique) pour se connecter à un réseau local, vous pouvez cocher le bouton radio IP dynamique/statique pour activer cette connexion secondaire.

  • Désactivé - La connexion secondaire est par défaut désactivée. Vous ne disposez donc que d'une connexion PPoE. Ce paramétrage est recommandé.
  • IP dynamique - Utilisez l'adresse IP dynamique pour vous connecter au réseau local fourni par votre FAI.
  • IP statique - Utilisez l'adresse IP statique pour vous connecter au réseau local fourni par votre FAI. Adresse IP - Adresse IP - Saisissez l'adresse IP fournie par votre FAI pour la connexion secondaire. Cette adresse sert uniquement pour l'accès au réseau local de la connexion secondaire. Masque de sous-réseau - Saisissez le masque de sous-réseau fourni par votre FAI pour la connexion secondaire.

Connexion à la demande - Vous pouvez configurer le routeur pour qu'il se déconnecte d'Internet après une période spécifiée (Délai d'inactivité max.). Si votre session a expiré pour cause d'inactivité, l'option Connexion à la demande permet au routeur de rétablir automatiquement la connexion dès que vous tentez d'accéder à nouveau à Internet. Pour activer cette option, cliquez sur le bouton radio. Si vous souhaitez que votre connexion Internet soit active en permanence, saisissez 0 comme Délai d'inactivité max.

Remarque : Parfois la connexion ne peut être interrompue, même en ayant spécifié un délai d'inactivité max. (0 à 99 min), car certaines applications fonctionnent en arrière-plan.

Connexion automatique - Lance la connexion automatiquement après déconnexion du routeur. Pour activer cette option, cliquez sur le bouton radio.

Connexion par plage horaire - Vous pouvez configurer le routeur pour qu'il se (dé)connecte selon une plage horaire. Dans le champ Durée, saisissez l'heure de début et de fin de déconnexion, au format HH-MM.

Connexion manuelle - Vous pouvez configurer le routeur afin de (dés)activer manuellement la connexion. Après une période d'inactivité donnée (Délai d'inactivité max.), le routeur suspend la connexion Internet. Si vous tentez d'y accéder à nouveau, le routeur ne peut pas la rétablir automatiquement. Pour activer cette option, cliquez sur le bouton radio. Si vous souhaitez que votre connexion Internet soit active en permanence, saisissez 0 comme Délai d'inactivité max. Sinon, saisissez le nombre de minutes que vous souhaitez voir s'écouler avant que votre connexion Internet soit interrompue, à moins qu'une nouvelle liaison soit nécessaire.

Remarque : Parfois la connexion ne peut être interrompue, même en ayant spécifié un délai d'inactivité max. (0 à 99 min), car certaines applications fonctionnent en arrière-plan.

Remarque : La fonction Connexion par plage horaire n'est active qu'une fois l'heure du système configurée dans Outils système sur Paramètres de l'horloge.

Cliquez sur le bouton Connexion pour vous connecter immédiatement.

Cliquez sur le bouton Déconnexion pour vous déconnecter immédiatement.

Cliquez sur Avancé pour configurer les options avancées.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.
