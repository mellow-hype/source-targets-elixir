Aide Filtrage MAC sans fil

La fonction de Filtrage MAC sans fil permet de contrôler l'accès au point d'accès des stations sans fil grâce à des règles basées sur leurs adresses MAC.

  • Adresse MAC - Adresse MAC de la station sans fil dont vous souhaitez contrôler l'accès.
  • Description - Description simple de la station sans fil.
  • État - Deux choix possibles : Activé ou Désactivé.

Pour désactiver la fonction, cliquez sur le bouton Désactiver (réglage par défaut).

Pour paramétrer une entrée, cliquez sur Activer et suivez les instructions ci-dessous :
Dans un premier temps, vous devez décider si les stations sans fil spécifiées seront autorisées ou refusées d'accès. En fonction de votre choix, cliquez sur le bouton radio devant Autoriser les stations spécifiées dans la table de filtrage ou devant Refuser les stations spécifiées dans la table de filtrage.

Pour ajouter un règle de filtrage, cliquez sur Ajouter... et suivez les instructions ci-dessous :

  1. Saisissez l'adresse MAC dans ce champ. L'adresse MAC est au format XX-XX-XX-XX-XX-XX (X = tout chiffre hexadécimal). Par exemple : 00-0A-EB-B0-00-0B.
  2. Décrivez simplement la station sans fil dans le champ Description. Par exemple : Station sans fil A.
  3. État - Sélectionnez Activé ou Désactivé dans la liste déroulante État.
  4. Cliquez sur Enregistrer pour sauvegarder l'entrée.

Pour ajouter d'autres règles, répétez les étapes 1 à 4.

Modifier ou supprimer une entrée :

  1. Cliquez sur Modifier ou sur Supprimer dans la colonne Modifier de la table de filtrage d'adresses MAC.
  2. Saisissez la valeur souhaitée dans la page Ajouter ou modifier un filtrage d'adresses MAC sans fil, puis cliquez sur Enregistrer.

Le bouton Tout activer permet d'activer l'ensemble des entrées, Tout désactiver les désactivent toutes et Tout supprimer vide la table des règles de filtrage.

Cliquez sur le Suivant ou Précédent pour accéder à la page souhaitée.

Remarque : Si vous activez la fonction et sélectionnez Autoriser les stations spécifiées dans la table de filtrage, mais qu'aucune station active n'est sélectionnée, aucune station ne pourra accéder au point d'accès.
