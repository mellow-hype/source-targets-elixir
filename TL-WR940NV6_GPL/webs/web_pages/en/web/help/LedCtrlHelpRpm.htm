Aide DEL

Vous pouvez changer l'état des voyants et configurer le mode nuit sur cette page.

Etat des DELs - Cliquez sur le bouton pour allumer / éteindre tous les voyants.

Mode nuit - Une fois le mode nuit activé, les DELs seront éteintes durant la période définie sans que cela n'altère le fonctionnement ni les performances du routeur.

Pour activer le mode nuit

  1. Sélectionner Activer
  2. Spécifier la plage horaire pendant laquelle les DEL doivent être éteintes.
  3. Cliquer sur Sauvegarder.
