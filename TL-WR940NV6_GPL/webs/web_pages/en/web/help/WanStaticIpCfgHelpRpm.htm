Aide WAN

Type de connexion WAN :

Si votre FAI utilise un serveur DHCP, sélectionnez l'option IP dynamique.

Si votre FAI fournit une adresse IP statique ou fixe, un masque de sous-réseau, une passerelle et des paramètres DNS, sélectionnez l'option IP statique.

Si votre FAI fournit une connexion PPPoE, sélectionnez l'option PPPoE/Russia PPPoE.

Si votre FAI fournit une connexion par câble BigPond, sélectionnez l'option Câble BigPond.

Si votre FAI fournit une connexion L2TP, sélectionnez l'option L2TP/Russia L2TP.

Si votre FAI fournit une connexion PPTP, sélectionnez l'option PPTP/Russia PPTP.

Si vous ne savez pas quel type de connexion choisir, cliquez sur le bouton Détecter pour que le routeur détermine automatiquement la connexion Internet pour les serveurs et les protocoles. Le type de connexion s'affiche dès que le routeur détecte une connexion Internet active. Cette information est donnée à titre indicatif uniquement. Pour vous assurer du type de connexion fournie, consultez votre FAI. Le routeur peut détecter les types de connexion Internet suivants :

  • PPPoE/Russia PPPoE - Connexion PPPoE nécessitant un identifiant et un mot de passe.
  • IP dynamique - Connexion par attribution d'une adresse IP dynamique.
  • IP statique - Connexion par attribution d'une adresse IP statique.

Adresse IP - Saisissez l'adresse IP en notation décimale pointée fournie par votre FAI.

Masque de sous-réseau - Saisissez le masque de sous-réseau en notation décimale pointée fourni par votre FAI.

Passerelle par défaut - Saisissez la passerelle par défaut en notation décimale pointée fournie par votre FAI.

Taille MTU - La Valeur de la MTU (Maximum Transmit Unit, unité de transmission maximale ) normale pour la plupart des réseaux Ethernet est 1500 octets. Cependant, en fonction de votre FAI, il se peut que vous deviez la changer. Toutefois, cela est rarement nécessaire ; ne procédez à la modification que si cela est requis pour votre connexion FAI.

DNS primaire - Saisissez l'adresse IP DNS en notation décimale pointée fournie par votre FAI.

DNS secondaire - Saisissez une autre adresse IP DNS en notation décimale pointée fournie par votre FAI.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.
