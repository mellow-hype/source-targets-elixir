Aide Paramètres de contrôle parental

Grâce à la fonction pratique de contrôle parental, le routeur peut restreindre l'accès à certains sites Web et le temps de navigation de votre enfant sur Internet. Sur cette page, vous pouvez créer une règle.

  • Contrôle parental - Cochez Activer si vous souhaitez appliquer cette fonction, sinon cochez Désactiver.
  • Adresse MAC du PC parent - Dans ce champ, saisissez l'adresse MAC du PC de contrôle ou utiliser le bouton Copier au-dessus.
  • Adresse MAC du PC - Ce champ affiche l'adresse MAC de l'ordinateur qui gère le routeur. Si l'adresse MAC de votre adaptateur est enregistrée, cliquez sur le bouton Copier au-dessus pour la reporter dans le champ Adresse MAC du PC parent ci-dessus.
  • Description site Web - Description du site Web autorisé pour l'ordinateur contrôlé.
  • Programmation horaire - Période autorisée durant laquelle l'ordinateur contrôlé peut accéder à Internet. Pour plus de détails, reportez-vous à Contrôle d'accès > Programmation horaire.
  • Activer - Cochez cette case pour activer une entrée spécifique.
  • Modifier - Permet de modifier ou supprimer une règle existante.

Exemple : Vous souhaitez que l'ordinateur de votre enfant dont l'adresse MAC est 00-11-22-33-44-AA puisse accéder à www.google.com uniquement le samedi sans aucune restriction du PC parental dont l'adresse MAC est 00-11-22-33-44-BB. Vous devez procéder comme suit :

  1. À gauche de l'écran, cliquez sur le menu Contrôle parental pour accéder aux Paramètres de contrôle parental. Cochez Activer, puis entrez l'adresse MAC 00-11-22-33-44-BB dans le champ Adresse Mac du PC parent.
  2. À gauche de l'écran, cliquez sur Contrôle d'accès > Programmation horaire pour accéder aux Paramètres Programmation horaire. Cliquez sur Ajouter... pour créer une entrée avec comme Description programmation horaire : Programmation_1, Jour : Sam, Heure : Toute la journée - 24 h.
  3. À gauche de l'écran, cliquez sur le menu Contrôle parental pour accéder aux Paramètres de contrôle parental.
 
  • Cliquez sur Ajouter....
  • Saisissez 00-11-22-33-44-AA dans le champ Adresse MAC du PC enfant.
  • Saisissez Autoriser Google dans le champ Description du site Web.
  • Saisissez www.google.com dans le champ Nom de domaine autorisé.
  • Dans la liste déroulante Horaires effectifs, sélectionnez l'entrée Programmation_1 que vous venez de créer.
  • Dans État, sélectionnez Activer.
  • Cliquez sur Enregistrer pour terminer.

Revenez ensuite sur la page Paramètres de contrôle parental, la liste suivante s'affiche :

ID Adresse MAC Description site Web Programmation horaire État Modifier
1 00-11-22-33-44-AA Autoriser Google Schedule_1 Modifier/Supprimer

Cliquez sur Ajouter... pour créer un Contrôle parental.

Cliquez sur le bouton Tout activer pour activer toutes les règles de la liste.

Cliquez sur le bouton Tout désactiver pour désactiver toutes les règles de la liste.

Cliquez sur Tout Supprimer pour Supprimer toutes les entrées de la table.

Cliquez sur le bouton Suivant ou Précédent pour accéder à la page souhaitée.
