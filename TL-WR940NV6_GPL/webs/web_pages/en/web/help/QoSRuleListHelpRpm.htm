Aide Liste des règles de contrôle de bande passante

Cette page affiche la liste des règles de contrôle de bande passante.

  • ID - Séquence d'entrée.
  • Description - Les informations descriptives comprennent les plages d'adresses et de ports et le protocole de couche de transport.
  • Bande passante sortante - Débit montant max. sur le port WAN ; la valeur par défaut est 0.
  • Bande passante entrante - Débit descendant max. sur le port WAN ; la valeur par défaut est 0.
  • Activer - État de la règle ; indique si elle est active ou non.
  • Modifier - Pour modifier ou supprimer une entrée existante.
