Aide Réservation d'adresse

Si vous spécifiez une adresse IP réservée à un ordinateur du réseau local, celui-ci recevra toujours la même adresse IP pour accéder au serveur DHCP. Les adresses IP réservées peuvent être assignées aux serveurs nécessitant l'allocation d'adresses IP permanentes.

  • Adresse MAC - Adresse MAC de l'ordinateur auquel vous souhaitez réserver une adresse IP.
  • Adresse IP réservée - Adresse IP réservée au périphérique.
  • État - Affiche l'état de l'entrée : actif ou inactif.
  • Modifier - Permet de modifier ou de supprimer une entrée.

Pour réserver des adresses IP, procédez comme suit :

  1. Saisissez l'adresse MAC (format : XX-XX-XX-XX-XX-XX) et l'adresse IP (en notation décimale pointée) de l'ordinateur que vous souhaitez ajouter.
  2. Cliquez sur Enregistrer.

Pour modifier une adresse IP réservée, procédez comme suit :

  1. Sélectionnez l'adresse réservée et éditez-la. Pour l'effacer, cliquez sur le lien Supprimer.
  2. Cliquez sur Enregistrer.

Cliquez sur le bouton Ajouter... pour créer une adresse de réservation.

Cliquez sur Tout activer pour activer toutes les entrées de la table.

Cliquez sur Tout désactiver pour désactiver toutes les entrées de la table.

Cliquez sur Tout Supprimer pour Supprimer toutes les entrées de la table.

Cliquez sur le bouton Suivant ou Précédent pour accéder à la page souhaitée.

Remarque : Les changements seront actifs une fois le périphérique redémarré.
