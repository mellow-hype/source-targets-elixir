Aide Paramètres DHCP

Ce périphérique est configuré par défaut en serveur DHCP (Dynamic Host Configuration Protocol) afin de fournir une configuration TCP/IP à tous les ordinateurs du réseau local connectés au périphérique.

  • Serveur DHCP - Pour Activer ou Désactiver le serveur. Si vous désactivez le serveur, votre réseau doit disposer d'un autre serveur DHCP. Vous devrez sinon configurer manuellement l'adresse IP de l'ordinateur.
  • Adresse IP de départ - Ce champ indique la première adresse du pool d'adresses IP. L'adresse IP de départ par défaut est 192.168.0.100.
  • Adresse IP de fin - Ce champ indique la dernière adresse du pool d'adresses IP. L'adresse IP de fin par défaut est 192.168.0.199.
  • Bail de l'adresse - Cette option définit la durée pendant laquelle un utilisateur réseau est autorisé à se connecter au périphérique avec l'adresse DHCP en cours. Saisissez la durée du bail de l'adresse DHCP en minutes. L'intervalle va de 1 à 2880 minutes. La valeur par défaut est de 120 minutes.
  • Passerelle par défaut - (facultatif) Saisissez l'adresse IP du port LAN de ce périphérique. La valeur par défaut est 192.168.0.1.
  • Domaine par défaut - (facultatif) Saisissez le nom de domaine de votre réseau.
  • DNS primaire - (facultatif) Saisissez l'adresse IP DNS fournie par votre FAI. Ou contactez votre FAI.
  • DNS secondaire - (facultatif) Vous pouvez saisir l'adresse IP d'un autre serveur DNS si votre FAI fournit deux serveurs DNS.

Remarque : Pour utiliser la fonction de serveur DHCP de ce périphérique, vous devez configurer tous les ordinateurs du réseau en mode « Obtenir une adresse IP automatiquement ». Cette fonction est activée une fois le périphérique redémarré.

Cliquez sur Enregistrer pour valider les modifications.
