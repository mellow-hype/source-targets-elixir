Aide Paramètres de l'horloge

Cette page vous permet de régler l'heure manuellement ou de configurer la synchronisation horaire automatique. Le routeur peut procéder à une mise à jour automatique de l'heure par un serveur NTP via Internet.

Fuseau horaire - Sélectionnez votre fuseau horaire dans la liste déroulante.

Réglage manuel de l'heure :

  1. Sélectionnez votre fuseau horaire.
  2. Saisissez la Date au format mois/jour/année.
  3. Saisissez l'heure au format heure/minutes/secondes.
  4. Cliquez sur Enregistrer.

Synchronisation horaire automatique :

  1. Entrez l'adresse ou le domaine du Serveur NTP 1 ou du Serveur NTP 2.
  2. Cliquez sur le bouton Obtenir GMT pour synchroniser l'heure.

Réglage de l'heure d'été :

  1. Sélectionnez Activer l'heure d'été pour activer cette fonction.
  2. Sélectionnez les dates de Début et de Fin de l'heure d'été.
  3. Cliquez sur Enregistrer.

Remarque :

  1. Ces paramètres sont utiles à certaines fonctions basées sur le temps, telles que celles du pare-feu. Ces fonctions ne peuvent être actives si l'heure n'est pas réglée. Par conséquent, il est important de définir les paramètres horaires dès que vous êtes connecté(e) au routeur.
  2. Le réglage de l'heure sera perdu si le routeur est éteint.
  3. Le routeur obtient automatiquement l'heure GMT via Internet s'il a été configuré correctement.
  4. Pour la configuration de l'heure d'été, la date de début doit être antérieure à la date de fin.
