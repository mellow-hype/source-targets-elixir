Aide Créer une cible d'accès

Sur cette page, vous pouvez ajouter une entrée à la liste de cibles.

  • Description cible - Dans ce champ, décrivez la cible. Important : Cette description doit être unique.
  • Mode - Deux options sont disponibles : Adresse IP et Nom de domaine. Sélectionnez celle de votre choix dans la liste déroulante.

S'il s'agit de l'Adresse IP, les éléments suivants s'affichent :

  • Adresse IP - Saisissez l'adresse IP (ou la plage d'adresses) de la cible (ou des cibles) en notation décimale pointée, ex. : 192.168.1.23.
  • Port cible - Indiquez le port ou la plage de ports pour la cible. Pour certains ports de service standard, vous pouvez utiliser le Port de service standard ci-dessous.
  • Protocole - Quatre options sont disponibles : Tous, TCP, UDP et ICMP. Faites votre choix dans la liste déroulante pour la cible.
  • Port de service standard - Il s'agit d'une liste des ports de service standard. Sélectionnez-en-un dans la liste déroulante. Le numéro de port correspondant s'affiche automatiquement dans le champ Port cible. Exemple : si vous sélectionnez « FTP », le numéro « 21 » s'affiche automatiquement dans Port cible.

Si Nom de domaine est sélectionné, les éléments suivants s'affichent :

  • Nom de domaine - Vous pouvez entrer quatre noms de domaines, en saisissant le nom complet ou simplement des mots clés (ex. : google). Tout nom de domaine contenant ces mots clés (www.google.com, www.google.cn) sera bloqué ou autorisé.
