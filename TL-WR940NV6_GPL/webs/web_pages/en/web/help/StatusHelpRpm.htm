Aide État

La page État affiche l'état du routeur et sa configuration actuelle. Toutes ces informations sont en lecture seule.

LAN - Les paramètres suivants concernent le port LAN du routeur. Vous pouvez les configurer sur la page Réseau -> LAN.

  • Adresse MAC - Adresse physique du routeur, côté LAN.
  • Adresse IP - Adresse IP LAN du routeur.
  • Masque de sous-réseau - Masque de sous-réseau associé à l'adresse IP LAN.

Sans fil - Paramètres ou informations sur le réseau sans fil. Vous pouvez configurer ces paramètres sur la page Sans fil -> Paramètres Sans fil.

  • Radio sans fil - Indique si la fonction Radio sans fil du routeur est activée ou non.
  • Nom (SSID) - SSID du routeur.
  • Mode - Mode de fonctionnement sans fil du routeur.
  • Largeur du canal - Bande passante du canal sans fil.
  • Canal - Canal sans fil utilisé.
  • Adresse MAC - Adresse physique du routeur, côté WLAN.

WAN - Les paramètres suivants concernent les ports WAN du routeur. Vous pouvez les configurer sur la page Réseau -> WAN.

  • Adresse MAC - Adresse physique du port WAN, côté Internet.
  • Adresse IP - Adresse IP WAN (Internet) actuelle. En cas d'adresse IP dynamique et en l'absence de connexion Internet, ce champ est vide ou contient « 0.0.0.0 ».
  • Masque de sous-réseau - Masque de sous-réseau associé à l'adresse IP WAN.
  • Passerelle par défaut - Passerelle utilisée par le routeur. Si vous utilisez un type de connexion Internet par IP dynamique, le bouton Renouveler s'affiche. Cliquez sur ce bouton pour obtenir dynamiquement les nouveaux paramètres IP de votre FAI. Si vous avez une adresse IP, le bouton Obtenir s'affiche. Cliquez sur ce bouton pour afficher l'adresse IP fournie par votre FAI.
  • Serveur DNS - Adresses IP des serveurs DNS utilisées par le routeur. La configuration d'adresses IP de plusieurs serveurs DNS est chose courante. En général, le premier serveur DNS disponible est utilisé.
  • Durée de connexion - Temps passé en ligne. Si vous utilisez une connexion WAN de typePPPoE, votre durée de connexion s'affiche ici. Cliquez sur le bouton Connexion ou Déconnexion pour vous (de)connecter d'Internet.

Connexion secondaire - Si vous utilisez un autre type de connexion fourni par votre FAI pour accéder à un réseau local, ces paramètres s'afficheront ici.

Statistiques de trafic - Statistiques de trafic du routeur.

  • Envoyé(s) (octets) - Octets envoyés du port WAN.
  • Envoyé(s) (paquets) - Paquets envoyés du port WAN.
  • Reçu(s) (octets) - Octets reçus du port WAN.
  • Reçu(s) (paquets) - Paquets reçus du port WAN.

Temps disponibilité syst. - Temps pendant lequel le routeur est disponible et en marche.

Cliquez sur le bouton Actualiser pour afficher l'état et les paramètres actuels du routeur.
