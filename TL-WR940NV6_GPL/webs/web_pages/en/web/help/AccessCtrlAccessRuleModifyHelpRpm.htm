Aide Modifier un contrôle d'accès à Internet

Vous pouvez modifier la règle sur cette page.

  • Nom de règle - Modifiez le nom de la règle dans ce champ. Important : Ce nom doit être unique.
  • Hôte - Vous pouvez modifier les champs suivants : Description hôte, Type d'adresse et Adresse IP LAN. Important : Toute modification entraîne l'écrasement de la règle précédente.
  • Cible - Vous pouvez modifier n'importe quelle information. Important : Toute modification entraîne l'écrasement de la règle précédente.
  • Programmation horaire - Vous pouvez modifier n'importe quelle information. Important : Toute modification entraîne l'écrasement de la règle précédente.
  • État - Deux options sont disponibles : Activer et Désactiver. Sélectionnez Activer pour appliquer la règle. Sélectionnez Désactiver pour qu'elle soit sans effet.

Cliquez sur Enregistrer pour terminer.

Cliquez sur Précédent pour revenir à la Gestion des règles de contrôle d'accès.
