Aide

Cette fonction vous permet d'ajouter rapidement un périphérique à votre réseau. Si le périphérique est compatible WPS (Wi-Fi Protected Setup) et doté d'un bouton de configuration, il vous suffit d'appuyer sur ce bouton, puis sur le bouton du périphérique pour l'ajouter à votre réseau. La DEL d'état du périphérique s'allume en vert pendant cinq minutes si ce dernier a pu être ajouté au réseau. Si le nouveau périphérique est compatible WPS et Code PIN, vous pouvez l'ajouter au réseau en saisissant son code PIN.

  • État - (Dés)activez ici la fonction.
  • PIN actuel - Affiche le code PIN du périphérique. La valeur par défaut est toujours indiquée sur l'étiquette ou dans le manuel d'utilisation.
  • Restaurer PIN - Pour restaurer le code PIN par défaut.
  • Gén. nouv. PIN - Cliquez sur ce bouton pour obtenir un nouveau code PIN aléatoire. Générer un nouveau code PIN garantit une sécurité accrue du réseau.
  • Désactiver PIN de ce périph. - Le registre externe WPS de saisie du code PIN de ce périphérique peut être (dés)activé manuellement. En cas de tentatives d'authentification erronées, la fonction est automatiquement désactivée.
  • Ajout de périphérique - Cliquez sur ce bouton pour ajouter manuellement un périphérique au réseau existant.

Remarque : La fonction ne peut être configurée si l'option Sans fil de ce périphérique est désactivée. Assurez-vous que la fonction Sans fil est bien activée avant de configurer le .
