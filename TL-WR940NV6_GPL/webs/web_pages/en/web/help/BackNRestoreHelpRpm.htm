Aide Sauvegarde et Restauration

Cliquez sur le bouton Sauvegarder pour enregistrer tous les paramètres de configuration de votre ordinateur local dans un fichier.

Pour restaurer la configuration du périphérique, suivez ces instructions :

  • Cliquez sur le bouton Parcourir pour rechercher le dossier de configuration que vous souhaitez restaurer.
  • Cliquez sur Restaurer pour mettre à jour la configuration à l'aide du fichier sélectionné.

Remarque : La configuration en cours sera remplacée par celle contenue dans le fichier. En cas d'erreur, le périphérique ne sera plus administré. Le processus de restauration dure environ 20 secondes, puis le périphérique redémarre automatiquement. Laissez le périphérique sous tension, afin d'éviter tout dommage.
