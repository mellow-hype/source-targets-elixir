Aide Sans fil

Point d'accès : mode de fonctionnement pour l'accès des stations sans fil.

  • Nom du réseau sans fil (SSID) : saisissez une chaîne de 32 caractères. Tous les périphériques sans fil du réseau devront utiliser ce nom (SSID). Le SSID par défaut permet d'assurer la sécurité de votre réseau sans fil. Nous vous recommandons cependant vivement de personnaliser ce SSID. Cette valeur est sensible à la casse. Par exemple, MONSSID est DIFFÉRENT de MonSsid.

  • Région : sélectionnez votre région dans la liste déroulante. Ce champ liste les régions où il est possible d'utiliser la fonction sans fil du point d'accès. Attention : Utiliser le point d'accès en mode sans fil dans une région non listée peut être illégal. Si votre pays ou région n'apparaît pas dans la liste, veuillez contacter l'administration locale compétente.

  • Canal : indique la fréquence de fonctionnement. Il n'est pas nécessaire de changer le canal sans fil, sauf en cas d'interférences constatées avec un autre point d'accès à proximité. En mode automatique, le point d'accès choisit automatiquement le meilleur canal.

Répéteur : En mode Répéteur, le répéteur sans fil relaie le signal entre ses stations et le point d'accès principal afin d'augmenter la portée sans fil.

  • Nom sans fil du point d'accès principal : Saisissez le nom du point d'accès distant (également appelé SSID) auquel vous souhaitez accéder. Cliquez sur le bouton Liste en dessous et sélectionnez un des résultats pour compléter ce champ.

  • Adresse MAC du point d'accès principal : Saisissez l'adresse MAC du point d'accès auquel vous souhaitez accéder. Si vous utilisez la fonction Liste, le champ Nom sans fil du point d'accès distant : est automatiquement rempli.

  • Liste : Cliquez sur ce bouton pour rechercher le point d'accès qui utilise le canal.
  • Mode WDS : Ce champ indique le mode WDS utilisé. Inutile de le changer, sauf en cas de problèmes de communication réseau avec le point d'accès principal. En mode automatique, le routeur choisit automatiquement le mode WDS adéquat.

Sélectionnez une des options de sécurité sans fil suivantes :

  • Aucune sécurité : La fonction de sécurité sans fil peut être (dés)activée. Si vous sélectionnez cette option, les stations sans fil pourront se connecter au périphérique sans chiffrement. Nous vous recommandons vivement de sélectionner une des options de sécurité suivantes.

  • Sécurité (WEP)

    • Type : Sélectionnez l'un des types suivants :

    • Automatique : Sélection automatique du type d'authentification ( Clé partagée ou Système ouvert) en fonction des capacités et demandes des stations sans fil.
    • Système ouvert : authentification par Système ouvert 802.11.
    • Clé partagée : authentification par Clé partagée 802.11.

    Format clé WEP : vous avez le choix entre les formats ASCII et Hexadécimal. Le format ASCII convient à toute combinaison de caractères de la longueur indiquée. Le format ASCII convient à toute combinaison de chiffres hexadécimaux (0-9, a-f, A-F) de la longueur indiqué.

    Paramètres clé WEP : choisissez la clé WEP parmi les quatre proposées en cliquant sur le bouton radio, puis entrez les informations sur la clé pour votre réseau. Toutes les stations clientes sans fil de votre réseau doivent utiliser les mêmes paramètres WEP que vous.

    Type clé : Choisissez la longueur de la clé WEP (64 bits, 128 bits ou 152 bits) pour le chiffrement.« Désactivé » signifie que la clé WEP est invalide.

    Pour le chiffrement 64 bits : vous pouvez saisir 10 chiffres hexadécimaux (toutes combinaisons de 0-9, a-f, A-F, hors clé nulle) ou 5 caractères ASCII.

    Pour le chiffrement 128 bits : vous pouvez saisir 26 chiffres hexadécimaux (toutes combinaisons de 0-9, a-f, A-F, hors clé nulle) ou 13 caractères ASCII.

    Pour le chiffrement 152 bits  : vous pouvez saisir 32 chiffres hexadécimaux (toutes combinaisons de 0-9, a-f, A-F, hors clé nulle) ou 16 caractères ASCII.

  • Sécurité renforcée (WPA/WPA2-PSK)

    • Version : Sélectionnez l'une des versions suivantes :

    • Automatique : Sélection automatique (WPA-Personnel ou WPA2-Personnel) en fonction des capacités et demandes des stations sans fil.
    • WPA-Personnel : Clé pré-partagée WPA.
    • WPA2-Personnel : Clé pré-partagée WPA2.

    Chiffrement : Vous avez le choix entre : Automatique, TKIP etAES.

    Mot de passe sans fil : Vous avez le choix entre ASCII ou Hexadécimal. En mode Hexadécimal, la clé doit comprendre entre 8 et 64 caractères. En mode ASCII, la clé doit comprendre entre 8 et 63 caractères.

    Intervalle de mise à jour de la clé de groupe : Indiquez l'intervalle de rafraîchissement en secondes. La valeur doit être comprise entre 0 et 30. La valeur 0 désactive la mise à jour.

  • Pas de modification : Si vous sélectionnez cette option, la configuration de la sécurité sans fil ne sera pas modifiée.
