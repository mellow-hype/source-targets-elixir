Aide État IPv6

La page État IPv6 affiche l'état et la configuration en cours d'utilisation du protocole IPv6 du routeur. Toutes ces informations sont en lecture seule.

    WAN
  • Type de connexion : connexion IPv6 pour WAN.
  • Adresse IPv6 : Adresse WAN IPv6.
  • Passerelle par défaut IPv6 : Passerelle par défaut du routeur.
  • DNS IPv6 primaire : Adresse DNS IPv6 primaire.
  • DNS IPv6 secondaire : Adresse DNS IPv6 secondaire.

    • LAN

  • Type d'assignation d'adresse IPv6 : Mode d'attribution de l'adresse IPv6 par le routeur pour un ordinateur en LAN, RADVD (Router Advertisement Daemon) et sur serveur DHCPv6 (Dynamic Host Configuration Protocol for IPv6).
  • Adresse IPv6 : Adresse IPv6 globale du réseau locale du routeur
  • Adresse Lien-Local : Adresse Lien-Local LAN du routeur.
