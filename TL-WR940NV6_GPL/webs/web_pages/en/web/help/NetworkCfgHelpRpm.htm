Aide Réseau local

Sur cette page, vous pouvez configurer les paramètres IP du réseau local.

  • Adresse MAC - Adresse physique des ports LAN, côté LAN. Cette valeur ne peut être modifiée.
  • Adresse IP - Saisissez l'adresse IP de votre routeur en notation décimale pointée (adresse d'usine par défaut : 192.168.0.1).
  • Masque de sous-réseau - Code d'adresse qui détermine la taille du réseau. Le code habituellement utilisé est 255.255.255.0.
  • Proxy IGMP - Si vous souhaitez regardez la télévision par IGMP, veuillez activer cette option.

Remarque :

  1. Si vous changez l'adresse IP LAN, vous devrez utiliser la nouvelle adresse IP pour vous connecter au routeur.
  2. Si la nouvelle adresse IP LAN n'appartient pas au même sous-réseau que la précédente, le pool d'adresses IP du serveur DHCP se configure automatiquement. Cependant, le serveur virtuel et l'hôte DMZ ne seront actifs qu'une fois reconfigurés.

Cliquez sur Enregistrer pour sauvegarder vos paramètres.
