Aide Créer un hôte

Sur cette page, vous pouvez ajouter une entrée à la liste d'hôtes.

  • Description hôte - Dans ce champ, donnez une description unique à l'hôte.
  • Mode - Deux options sont disponibles : Adresse IP et Adresse MAC. Faites votre choix dans la liste déroulante.

S'il s'agit d'Adresse IP, les éléments suivants apparaissent :

  • Adresse IP LAN - Entrez l'adresse IP ou la plage d'adresses de l'hôte en notation décimale pointée, ex. : 192.168.0.23.

Si vous sélectionnez Adresse MAC, les éléments suivants apparaissent :

  • Adresse MAC - Saisissez l'adresse MAC de l'hôte au format XX-XX-XX-XX-XX-XX (ex. : 00-11-22-33-44-AA).

Cliquez sur Suivant pour continuer ou sur Précédent pour revenir à la page précédente.
