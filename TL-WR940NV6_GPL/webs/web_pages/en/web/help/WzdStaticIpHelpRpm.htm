Aide IP statique

Votre FAI vous a normalement fourni les paramètres IP.

Adresse IP - Adresse IP WAN vue par les utilisateurs externes sur Internet (y compris votre FAI). Saisissez l'adresse IP dans ce champ.

Masque de sous-réseau - Le masque de sous-réseau est utilisé pour l'adresse IP WAN (255.255.255.0 en général).

Passerelle par défaut - Saisissez la passerelle par défaut dans ce champ.

DNS primaire - Saisissez l'adresse IP DNS dans ce champ.

DNS secondaire - Si votre FAI vous a fourni une seconde adresse IP DNS, saisissez-la dans ce champ.
