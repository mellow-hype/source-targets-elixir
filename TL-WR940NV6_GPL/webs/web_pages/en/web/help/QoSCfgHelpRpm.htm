Aide Paramètres de contrôle de bande passante

Sur cette page, vous pouvez (dés)activer la fonction de Contrôle de bande passante. Cette fonction doit être activée pour que les règles de contrôle de bande passante s'appliquent correctement.

  • Activer le contrôle de bande passante - Si cette fonction est activée, les règles de contrôle de bande passante s'appliquent.
  • Type de ligne - Veuillez sélectionner le type de connexion utilisée afin que le module de contrôle de bande passante puisse fonctionner correctement
  • Bande passante sortante - Débit montant sur le port WAN.
  • Bande passante entrante - Débit descendant sur le port WAN.
