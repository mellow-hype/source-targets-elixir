Aide Serveurs virtuels

Les serveurs virtuels peuvent servir à définir des services publics sur le réseau local. Un serveur virtuel est un port de service qui redirige toutes les requêtes reçues d'Internet vers un ordinateur spécifié par l'IP du serveur. Tout ordinateur faisant office de serveur virtuel doit avoir une adresse IP statique ou réservée, car son adresse IP peut changer lors de l'utilisation de la fonction DHCP.

  • Port de service - Numéros des ports de service externes. Vous pouvez saisir un port ou une plage de ports de service au format XXX - YYY, où XXX est le port de départ et YYY le port de fin.
  • Port interne - Numéro du port de service interne de l'ordinateur qui exécute l'application. Vous pouvez entrer un numéro de port spécifique ou laisser ce champ vierge si le Port interne est identique au Port de service.
  • Adresse IP - Adresse IP de l'ordinateur qui exécute l'application.
  • Protocole - Protocole utilisé pour l'application : TCP, UDP ou Tous (tous les protocoles pris en charge par le routeur).
  • État - État de l'entrée, « Activé » signifiant que le serveur virtuel est activé.
  • Port de service standard - Certains services standard existent déjà dans la liste déroulante.
  • Modifier - Permet de modifier ou de supprimer une entrée.

Définir un serveur virtuel :

  1. Cliquez sur le bouton Ajouter....
  2. Sélectionnez le service que vous souhaitez utiliser dans la liste Port de service standard. Si le menu Port de service standard ne propose pas le service recherché, saisissez le numéro du port ou la plage de ports dans le champ Port de service.
  3. Saisissez l'adresse IP de l'ordinateur qui exécute l'application dans le champ Adresse IP.
  4. Dans la liste déroulante, sélectionnez le protocole utilisé pour l'application : TCP, UDP ou Tous.
  5. Sélectionnez Activé dans la liste déroulante du champ État.
  6. Cliquez sur Enregistrer.

Remarque : Il est possible que votre ordinateur ou serveur dispose de plusieurs types de service. Dans ce cas, sélectionnez un autre service et entrez la même adresse IP de l'ordinateur ou du serveur.

Modifier ou supprimer une entrée :

  1. Recherchez l'entrée souhaitée dans la table.
  2. Cliquez sur Modifier ou Supprimer dans la colonne Modifier.

Cliquez sur le bouton Tout activer pour activer toutes les entrées.

Cliquez sur le bouton Tout désactiver pour désactiver toutes les entrées.

Cliquez sur le bouton Tout Supprimer pour effacer toutes les entrées.

Cliquez sur Suivant ou Précédent pour accéder à la page souhaitée.
