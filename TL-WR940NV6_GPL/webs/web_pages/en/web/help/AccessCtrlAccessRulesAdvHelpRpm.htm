Aide Ajouter ou Modifier un contrôle d'accès à Internet

Vous pouvez créer une règle sur cette page.

  • Règle - Dans ce champ, donnez un nom à la règle. Important : Ce nom doit être unique.
  • Hôte - Sélectionnez un hôte dans la liste déroulante.
  • Cible - Sélectionnez une cible dans la liste déroulante. La valeur par défaut est N'importe quelle cible.
  • Programmation horaire - Sélectionnez une programmation horaire dans la liste déroulante. La valeur par défaut est Aucune.
  • État - Deux options sont disponibles : Activer et Désactiver. Sélectionnez Activer pour appliquer la règle. Sélectionnez Désactiver pour qu'elle soit sans effet.

Cliquez sur Enregistrer pour terminer.

Cliquez sur Précédent pour revenir à la Gestion des règles de contrôle d'accès.
